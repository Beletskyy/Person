# Advanced Backbone + Spring Boot

## How to run?
```
mvn spring-boot:run
```
Server will be able on localhost:3000

## Features:
    Spring Boot + JPA + H2
    Backbone + plugins
    DustJS
    RequireJS
    Jquery + Jquery-ui
    Lodash
    NPM
    Bower

## Description:

   - A simple list of contacts with CRUD operations.
   - Validation.
   - Confirmation pop-up before deleting user.
